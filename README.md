# 网页自动滚动助手 (Auto Scroll Pro) - 使用手册

网页自动滚动助手是一款为 Chrome / Edge 浏览器量身定制的高性能、高颜值自动滚动插件。它支持平滑微步滚动与定距翻页滚动，配备优雅的毛玻璃暗色面板，且支持全局快捷键快捷启停。

---

## 📁 项目文件清单
在本项目目录中包含以下核心文件：
1. `manifest.json`：声明 Manifest V3 扩展配置、权限（存储、脚本注入、活动标签）及快捷键。
2. `popup.html` & `popup.css`：基于 CSS 变量和毛玻璃拟物化设计的精美控制面板。
3. `popup.js`：控制面板的交互与状态存取逻辑。
4. `content.js`：注入到网页中的高精度滚动执行引擎，采用 `requestAnimationFrame` 驱动。
5. `background.js`：后台服务进程，负责全局快捷键侦听与动态角标更新。
6. `icon16.png`, `icon48.png`, `icon128.png`：插件的发光双折线图标。
7. `generate_icons.py`：自动绘制生成 PNG 图标的 Python 脚本。

## 🛠️ 安装方法

1. 打开 Edge 或 Chrome 浏览器，进入扩展管理页面（地址栏输入 `edge://extensions/` 或 `chrome://extensions/`）。
2. 在页面右上角开启 **“开发者模式”**。
3. 点击左上角的 **“加载已解压的扩展程序”**。
4. 在文件选择框中选择您所下载/克隆的项目文件夹（即包含 `manifest.json` 的目录）。
5. 载入成功后，建议点击浏览器右上角的 **拼图图标 🧩**，将本插件 **固定到工具栏** 📌，以便随时查看运行状态。

## 🎮 功能与操作指南

### 1. 双系统支持与启停控制
* **双系统兼容**：本插件完美支持 Windows 与 macOS。插件控制面板底部会根据您的操作系统，动态自适应地显示正确的快捷键提示。
* **面板操作**：点击工具栏插件图标打开面板，点击中心的 **播放/暂停** 按钮。
* **键盘快捷键**：在任何网页上直接按下 **`⌥ Option + S`** (Mac) 或 **`Alt + S`** (Windows) 即可一键启停，无需打开面板。

### 2. 滚动模式
* **平滑滚动 (Smooth Scroll)**：
  * 以每秒设定的像素值（px/s）进行微步平滑滚动，完美适合边看边读。
  * 范围：`1 px/s` 到 `150 px/s`。
* **定距滚动 (Interval Scroll)**：
  * 每隔指定秒数，向下瞬时滚动指定屏数（基于当前视口高度）。
  * 例如：每隔 `3 秒` 滚动 `0.5 屏`。

### 3. 触底检测与自动暂停
* 开启“触底自动停止”后，当滚动至页面底部时，插件会自动暂停并重置状态，同时工具栏图标上会短暂显示 **“完成”** 蓝色角标。

## ⚠️ 注意事项与权限说明
* **浏览器自带页面限制**：根据浏览器安全策略，任何扩展插件都无法在浏览器内置页面（如 `edge://*`、`chrome://*` 等）中运行。在这些页面上，控制面板会自动显示“已禁用”。
* **文件权限**：若需在本地 HTML 文件 (`file:///*`) 上使用，请在浏览器的本插件详情页中开启 **“允许访问文件 URL”**。
